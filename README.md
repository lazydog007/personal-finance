# Personal Finance Service

This backend service, built using Python and Flask, allows you to track your spending, manage your budget, and monitor your savings effectively.

## Features

- Track your spending by category
- Set and manage your budget for different categories
- Monitor your savings progress towards your financial goals

## Technologies Used

- Python
- Flask
- SQLite

## Prereqs

## Installation

1. Clone the repository
2. Install dependencies using `pipenv install`
3. Set up your SQLite database
4. Run the server using `python app.py`

## API Endpoints

- GET /transactions - Get all transactions
- POST /transactions - Add a new transaction
- GET /budgets - Get all budget categories
- POST /budgets - Add a new budget category
- GET /savings - Get all savings goals
- POST /savings - Add a new savings goal

## Usage

- Use the provided API endpoints to interact with the service
- Track your transactions, set budgets, and manage your savings effectively
